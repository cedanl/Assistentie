# Frontend

Streamlit frontend voor de Edupulse applicatie.

## Functionaliteit
- Dashboard met studentenoverzicht en filters (opleiding, klas, mentor)
- Visualisaties: cijferverdeling, aanwezigheid per opleiding
- AI-powered Q&A over studentendata
- Managementsamenvatting genereren
- Uitvalrisico voorspellen met SHAP-uitleg per student

## Belangrijke bestanden
- `app.py` - Hoofdapplicatie met Streamlit UI
- `ui.py` - UI componenten (momenteel niet gebruikt)

## Starten
```bash
streamlit run frontend/app.py
```
