# CEDA Edupulse App

Een educatieve applicatie ontwikkeld door CEDA met een FastAPI backend en Streamlit frontend.

## Installatie

Installeer de dependencies met uv of pip:

```bash
uv sync
# of
pip install -r requirements.txt
```

## Gebruik

Start de applicatie in twee stappen:

1. Start de FastAPI backend:
   - Windows: `1_start_fastapi.bat`
   - Linux/Mac: `./1_start_fastapi.sh`

2. Start de Streamlit frontend:
   - Windows: `2_start_streamlit.bat`
   - Linux/Mac: `./2_start_streamlit.sh`

## Technologieën

- FastAPI - Backend API
- Streamlit - Frontend interface
- OpenAI - AI integratie
- Pandas & Plotly - Data analyse en visualisatie
- scikit-learn & SHAP - Machine learning
